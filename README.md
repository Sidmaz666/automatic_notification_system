# automatic_notification_system
A Simple website to send customized automatic scheduled Notifications to a given Email.

## Front-End Preview
<img src="https://user-images.githubusercontent.com/81381178/164507757-55622ff7-e6ff-454a-8783-e871841cb570.png" width="100%">

## ENV Variables
1. `DB_URI` - Mongo DB URI 
2. `EMAIL` - Admin Email Address
3. `PASS` - Admin Email Password

## Modules
1. dotenv
2. ejs
3. express
4. mongoose
5. nodemailer

## Available Commands/Scripts
1. `start`: `node server.js`,
2. `devStart`: `nodemon server.js`,
3. `watch:css`: `postcss ./public/css/tailwind.css -o ./public/css/style.css --watch`
